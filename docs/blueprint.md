# **App Name**: Dynamic Portfolio

## Core Features:

- Animated Hero Section: Display the developer's name and role with an animated entrance effect.
- Smooth Scrolling: Implement smooth scrolling across different sections of the portfolio for enhanced UX.
- Interactive Project Cards: Showcase projects with interactive cards that elevate and display project details on hover with a subtle 3D rotation.
- Animated Skills Section: Visualize skills with animated progress bars that fill up on scroll into view, accompanied by animated tooltips.
- Morphing Page Transitions: Create fluid, morphing transitions between different pages or sections.

## Style Guidelines:

- Background color: Off-white (#F9FAFB) to provide a clean and modern base.
- Primary color: Vibrant orange (#FF5722) for interactive elements and highlights.
- Accent color: Bright blue (#2196F3) for secondary interactive elements, creating contrast and visual interest.
- Body and headline font: 'Inter', a sans-serif font, for a modern and legible design.
- Preloader animation: Abstract shapes in orange and blue intertwine and fade out.
- Subtle hover effects: Elements react with scale changes and minimal color waves upon interaction.
